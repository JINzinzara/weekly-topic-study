## Topic 1. YOLO(You Only Look Once) 모델의 주요 특징과 장점

### 1. 객체 탐지(Object Detection)
- **정의:** 이미지 내에 존재하는 객체를 찾고, 구분하는 기술이다
- **방법:** 객체가 있는 영역을 Bounding Box로 표시하고, 그 안에 존재하는 객체를 클래스(label)로 분류한다
- **사례:** 이미지 인식, 자율 주행 자동차 등


### 2. 내용

#### YOLO란?
- **정의:** "You Only Look Once"로 이미지를 한번 보고, 바로 객체를 인식하는 딥러닝 기술
- **등장 배경:** R-CNN은 객체를 인식하고 해당하는 클래스로 분류하기까지 두 단계에 걸쳐서 동작하였기 때문에, 시간 소요가 많음 &rarr; YOLO는 두 단계를 한번에 처리 하기 때문에 실시간 물체 인식 가능하다

#### 작동 방식
- 1. 입력 이미지를 7x7 격자로 나눈 뒤, 각 격자 셀별로 2개의 bounding box를 예측
- 2. 예측된 bounding box를 NMS과정을 통해 가장 높은 클래스 확률값을 가진 예측 결과(bounding box)만 남는다
*NMS(Non-Maximun Suppression): 한 물체에 겹쳐 나온 박스들 중 점수 높은 것만 남기고 나머지를 제거*

#### 주요 특징
| 특징 | 설명 |
|------|------|
| 단일 신경망 | 이미지 전체를 하나의 신경망으로 처리|
| 실시간 처리 | 초당 수십~수백 프레임 처리 가능 |
| 그리드 기반 예측 | 이미지를 n×n 격자로 나눠 각 셀에서 객체 예측 |
| Bounding Box 예측 | 각 격자 셀에서 박스 위치(x, y, w, h)와 신뢰도를 동시에 예측 |

#### YOLO vs 기존 객체 탐지 모델 비교
| 구분 | YOLO | R-CNN 계열 |
|------|------|-----------|
| 처리 방식 | One-stage | Two-stage |
| 속도 | 빠름(실시간) | 느림 |
| 정확도 | 상대적으로 낮음 | 높음 |

### 3. 왜 중요한가?
- 빠른 속도가 필요한 실시간 물체 검출 수행에 용이하다

## Topic 2. mAP(mean Average Precision)의 개념과 활용

### 1. 핵심 개념 요약
mAP는 모든 클래스에 대한 AP값을 평균낸 모델 성능 평가 지표이며,
모델이 객체를 정확한 box 위치로 클래스(label)을 분류하였는지 평가할 때 사용된다.
높은 mAP 점수 일수록 해당 모델이 객체를 잘 인지하였는지를 알 수 있기 때문에 안전과 효율성을 중시하는 테스크에서 많이 사용된다.

### 2. 상세 내용

#### AP (Average Precision)
- **정의:** 정보 검색 및 객체 탐지와 같은 딥러닝 모델의 성능을 평가하는 지표
- **계산 방법:** Precision-Recall 곡선 아래 부분의 넓이(0~1값)

#### mAP(mean Average Precision)
- **정의:** 모든 클래스에 대한 AP의 평균값
- **왜 사용하나:** 모델이 얼마나 객체를 잘 찾는지 + 객체 주변에 얼마나 정확한 bounding box를 위치시키는지 평가

#### 핵심 개념 정리

##### Precision & Recall
| 개념 | 정의 | 수식 |
|------|------|------|
| Precision (정밀도) | 모델이 탐지한 모든 객체 중, 실제로 올바른 비율 | TP / (TP + FP) |
| Recall (재현율) | 이미지에 실제로 존재하는 객체 중, 모델이 찾은 비율 | TP / (TP + FN) |

##### IoU (Intersection over Union)
- **정의:** 모델이 박스를 예측하였을 때, 예측한 박스가 정답과 얼마나 잘 맞는지를 나타내는 지표
- **역할:** 예측 판정, NMS의 겹친 박스를 제거할 때의 겹침의 기준
- **임계값(Threshold) 기준:**
  - IoU=0.5 기본 평가 기준
  - IoU-0.75 정밀한 위치 탐지
  - mAP@0.5:0.95 0.5부터 0.95까지 0.05단위씩 증가한 평균값(COCO데이터셋 기준)

#### mAP 계산 흐름
- IoU 기준으로 TP / FP 판별
- 클래스별 Precision-Recall 곡선 생성
- 각 클래스의 AP 계산
- 전체 클래스 AP의 평균 → mAP

#### mAP 계산 방식
| mAP 범위 | 의미 |
|---------|------|
| mAP@0.5 | IoU를 0.5이상일 때 탐지가 정확하다고 간주하고 계산한 mAP |
| mAP@0.5:0.95 | IoU를 0.05에서 0.95 사이를 0.05단위로 평균낸 mAP |

#### mAP 값 해석
| mAP 범위 | 의미 |
|---------|------|
| 0.5 이상 | 탐지를 잘하였음을 나타내는 기본적인 값 |
| 0.75 이상 | 우수한 성능으로 객체를 탐지 |

### 3. mAP 관련 지표 비교
| 지표 | 사용 모델 |
|-----|---------|
| mAP | 객체 인식 |
| Accuracy | 이미지 분류 |

### 4. 왜 중요한가?
- 높은 mAP 점수는 안전과 효율성이 중요한 환경에서 사용 됨(자율 주행 기술, 의료 이미지 분석)
