## Topic 1. 결정 트리의 장단점 & 부스팅 앙상블 기법

### 1. 상세 내용

#### 결정 트리 (Decision Tree)
- **정의:** 질문을 반복해서 데이터를 분류하는 모델(비모수 지도 학습 알고리즘)
- **작동 방식:** 많은 질문들 중에서 가장 높은 점수의 질문(정보 이득/잎의 순도)으로 데이터를 두 그룹으로 나눈다. 더이상 나눌 필요가 없을 때까지 해당 과정을 반복한다.

#### 결정 트리 장단점
| 구분 | 내용 |
|------|------|
| 장점 | 범주 및 연속형 수치를 모두 예측 가능 |
| 장점 | 변수 간의 상호작용과 비선형성을 반영하기 때문에 정확한 예측 가능 |
| 장점 | 분석 결과 해석 쉬움(해석 가능성) |
| 장점 | 스케일링과 같은 전처리 필요 없음 |
| 단점 | 과적합의 문제 발생(가지치기로 해결) |
| 단점 | 데이터 분포가 일정하지 않거나 노이즈가 포함될 경우 예측 어려움 |

## Topic 2. 부스팅 모델 리서치

### 1. 핵심 개념 요약

#### 부스팅 (Boosting)
- **정의:** 여러 모델을 순차적으로 만들되, 이전 모델이 틀린 부분을 다음 모델이 집중적으로 학습하는 방식
- **핵심 아이디어:** 순차적, 보완

#### 배깅 vs 부스팅 비교
| 구분 | 배깅 (Bagging) | 부스팅 (Boosting) |
|------|--------------|----------------|
| 학습 방식 | 무작위/독립적 | 순차적/보완 |
| 대표 모델 | RandomForest | XGBoost |

### 2. 부스팅 모델 비교 정리

#### Gradient Boosting
- **특징:** 이전 예측에서의 오차를 순차적으로 보정
- **장점:** 예측 성능 좋음
- **단점:** 과적합, 속도 느림

#### XGBoost
- **특징:** Regularization, 자동 가지치기, GPU 지원
- **장점:** 속도 빠름, Regularization을 통한 과적합 방지, 분류/회귀 가능
- **단점:** 학습 데이터 내의 값만 예측 가능, 튜닝 하이퍼파라미터 많음

#### LightGBM
- **특징:** 트리의 균형보다 최대 손실 값을 가지는 leaf 노드를 지속적으로 분할(leaf-wise 트리 분할/비대칭적 트리)
- **장점:** categorical feature 자동 변환 및 최적 분할 가능, 메모리 사용향 적음
- **단점:** 작은 데이터셋에서는 과적합 취약ㅊ

#### CatBoost
- **특징:** level-wise 방식으로 트리를 형성하지만 feature을 모두 동일하게 대칭적인 트리 구조
- **장점:** GBM에 비해 과적합 적음, 범주형 변수에 대한 모델의 정확도와 속도 향상, 시계열 데이터 처리
- **단점:** 결측값 처리 X, 수치형 변수 학습 속도 느림

#### AdaBoost
- **특징:** 예측에 실패한 샘플이 다시 학습될 수 있도록 가중치 부가(잘못 예측한 데이터 분류에 집중)
- **장점:** 실수를 반복적으로 수정하여 정확도 향상, RF에 비해 학습속도 빠름
- **단점:** 노이즈 데이터/이상치에 민감, 하이퍼파라미터 복잡

## Topic 3. 주성분 분석(PCA)과 요인 분석(FA)의 차이

### 1. 상세 내용

#### 주성분 분석 (PCA, Principal Component Analysis)
- **정의:** 상관관계를 이용하여 이를 대표하는 주성분을 추출해, 차원을 축소(분산 보존)
- **목적:** 다중 상관성 및 과적합의 문제 해결
- **작동 방식:** 데이터를 축에 비추어 봤을 때, 가장 높은 분산 데이터의 축을 찾아 그 축으로 차원을 축소

#### 요인 분석 (FA, Factor Analysis)
- **정의:** 관찰된 여러 변수들 사이에 존재하는 공통된 요인(잠재 변수)을 찾아내는 통계 기법
- **목적:** 내재된 연관성을 탐색, 잠재 요인을 도출, 관찰 불가능한 공통 요인 추정
- **작동 방식:** 관찰된 변수의 공통적으로 영향을 받는 잠재 변수를 도출하고, 변수의 개수를 줄이며 데이터의 핵심적인 공통된 정보를 유지

#### PCA vs 요인 분석 비교
| 구분 | PCA | 요인 분석 |
|------|-----|---------|
| 목적 | 분산 보존 | 공통된 요인 추정 |
| 방향 | 데이터&rarr;주성분 | 잠재요인&rarr;데이터 |
| 결과 해석 | 해석 어려움 | 개념적 의미 부여 가능 |
| 사용 상황 | 차원 축소, 시각화, 전처리 | 심리/사회과학에서의 잠재 구조 탐색|
